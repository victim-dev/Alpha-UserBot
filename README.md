⚙️ Setup

1. Clone Repository

git clone https://github.com/YOUR_USERNAME/alpha-userbot.git
cd alpha-userbot

---

2. Install Requirements

pip install -r requirements.txt

---

3. Create ".env"

API_ID=
API_HASH=
SESSION_STRING=
OWNER_ID=

NVIDIA_API_KEY=
API_BASE=https://integrate.api.nvidia.com/v1
MODEL=minimaxai/minimax-m2.5

MONGO_URI=

---

4. Run

python bot.py
